About

The in-game model browser is buggy and hard to use. Use this site instead to find a model, then use it in game by typing a model command in console (example: "model betagordon"). You can click on a model's name to copy its model command to the clipboard.

You might notice models with a lot of polygons lag when you try to view them on this site. The same thing happens in game when lots of people use high-poly models. Try not to use a model with more than about 2,000 polygons if you play on a server with a high player count (20+). Models that let you choose colors also have a significant performance hit.

I want to add a model uploader here eventually, but until then I'm just gonna copy stuff from GameBanana every few months or so.

Model Names

A lot of people copy-paste models and then rename them. These models then get included in packs and distributed. Because of this, I don't know if any of the model names here are correct. Let me (w00tguy) know if a model name is wrong and I'll fix it.
